Steps Bot Code Running Q&A

Making your first discord.py bot! (But on Termux!!!)

  1. Download Termux: Playstore or GitHub
  2. Open Termux
  3. pkg update && upgrade -y
  4. pkg install python -y
  5. pip install -U discord discord.py

You've now completed the prerequisites!

Bot Code

import discord
from discord.ext import commands

intents = discord.Intents.default()
intents.message_content = True

bot = commands.Bot(command_prefix="!", intents=intents)

@bot.event
async def on_message(message):
    if message.author.bot:
        return
    if message.content.lower() == "ping":
        await message.channel.send("pong")
    await bot.process_commands(message)

bot.run("YOUR_TOKEN")

Running

Save with nano bot.py. Then Ctrl + X → Y → Enter

Run with python bot.py

Your bot is now live!
