Getting Started (But on termux!!!)

Follow the steps given below:-

  1. Download Termux if you haven't already: Playstore or GitHub
  2. Open Termux
  3. Execute command pkg update && upgrade -y it is optional but recommended.
  4. After that execute command pkg install python -y which installs python which is roughly 100mbs.
  5. After the installation of python is done execute command pip install -U discord discord.py

You've now completed the prerequisites!

Now for the scary part - Coding the bot, but not to worry here's the basic template:-

import discord
from discord.ext import commands

intents = discord.Intents.default()
intents.message_content = True

bot = commands.Bot(command_prefix="!", intents=intents)

@bot.event
async def on_message(message):
    if message.author.bot:
        return
    if message.content.lower() == "ping":
        await message.channel.send("pong")
    await bot.process_commands(message)

bot.run("YOUR_TOKEN")

But now, where to put the code???

Eh dont worry, just do nano bot.py to create a file called bot.py (you can change the name) and when it opens in termux just paste the code above. And then do ctrl+x then Y for yes and then press enter, How to press enter on mobile? breh it the button you use to create new line on your soft keyboard which you type with!

Now that's done, but how do I run the code? Simply do python bot.py (or whatever the file name is) 

And that's it, you've a bot running!
